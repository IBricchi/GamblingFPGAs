## web-server setup

Initial setup for installing the required dependencies.

```bash
 ./setup.sh
```

## Run server

### Setup

To run the server simply run.

```bash
  ./run.sh
```

### Web Files

http://127.0.0.1:5000/

http://127.0.0.1:5000/data/
